# biggan-deep-256 图片素材

由神经网络 biggan-deep-256 生成的图片

* 模型来源：https://tfhub.dev/deepmind/biggan-deep-256/1
* 使用 tensorflow 2.3 运行
* 共 1000 类，每类生成 100 张图片，以 jpg 格式储存
